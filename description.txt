========================================
       Описание работы программы
       «Симулятор кофемашины»
========================================

НАЗНАЧЕНИЕ
----------
Программа моделирует работу кофемашины: читает рецепты напитков из файла,
«готовит» каждый напиток, задействуя соответствующие части машины, и выводит
название напитка вместе с суммарным временем его приготовления.

АРХИТЕКТУРА
-----------
Проект построен по принципам SOLID и использует основные концепции ООП
(инкапсуляция, наследование, полиморфизм, абстракция).

Основные компоненты:

1. Иерархия ингредиентов (include/ingredients/)
   - Ingredient          — абстрактный базовый класс.
                           Хранит название, количество порций и начальную
                           температуру ингредиента (по умолчанию 24 °C).
   - WetIngredient       — мокрый ингредиент; знает время нагрева
                           (со скоростью 1 °C/сек до целевой температуры).
   - DryIngredient       — сухой ингредиент; знает время растворения.
   Конкретные ингредиенты:
   - CoffeeBean  — кофейные зёрна (особый ингредиент: сначала молются,
                   потом варятся).
   - Milk        — молоко (мокрый, нагревается до ~67 °C).
   - Cream       — сливки (мокрый, нагревается до ~67 °C).
   - Water       — вода (мокрый, нагревается до 100 °C).
   - Brandy      — бренди (мокрый, добавляется без нагрева).
   - Sugar       — сахар (сухой, требует времени на растворение).
   - VanillaSugar — ванильный сахар (сухой).
   - Cinnamon    — корица (сухой, не требует времени).
   - Syrup       — сироп (мокрый).
   - IngredientFactory — фабрика, создающая нужный объект ингредиента
                         по строковому имени из файла рецептов.

2. Иерархия частей кофемашины (include/machine/)
   - MachinePart    — абстрактный базовый класс для деталей машины.
   - Grinder        — кофемолка; обрабатывает только кофейные зёрна
                      (14–19 сек на помол).
   - BrewingGroup   — группа заваривания; варит молотый кофе
                      (19–24 сек).
   - Boiler         — бойлер; нагревает воду до 100 °C.
   - SteamNozzle    — паровая форсунка; нагревает молочные продукты
                      до 65–70 °C, а также растворяет сухие добавки.

3. Рецепты (include/recipe/)
   - Recipe           — хранит название напитка и список его ингредиентов.
   - RecipeRepository — читает файл data/recipes.txt и возвращает
                        вектор объектов Recipe.

4. CoffeeMachine (include/machine/CoffeeMachine.h)
   Для каждого рецепта параллельно задействует все части машины:
   - coffeeTime  = время кофемолки + время группы заваривания
   - boilerTime  = время бойлера
   - nozzleTime  = время форсунки
   Итоговое время приготовления = max(coffeeTime, boilerTime, nozzleTime),
   т.к. части машины работают одновременно.

5. Display (include/Display.h)
   Выводит результаты в консоль в виде:
       Название напитка
       N секунд
       ----------------------------------------

6. Точка входа (src/main.cpp)
   - Создаёт RecipeRepository, загружает рецепты из data/recipes.txt.
   - Создаёт CoffeeMachine.
   - Для каждого рецепта вызывает machine.prepare(recipe).
   - Передаёт все результаты в Display::show().

ФОРМАТ ФАЙЛА РЕЦЕПТОВ (data/recipes.txt)
-----------------------------------------
[Название напитка]
ингредиент1 <количество порций>
ингредиент2 <количество порций>
...

Пример:
  [Латте]
  кофе 1
  молоко 2

ПРИМЕР ВЫВОДА ПРОГРАММЫ
------------------------
========================================
        Кофемашина - Результаты
========================================
Латте
128 секунд
----------------------------------------
Капучино
147 секунд
----------------------------------------
...

СБОРКА И ЗАПУСК
---------------
Требования: компилятор C++17, CMake >= 3.10.

  mkdir build && cd build
  cmake ..
  cmake --build .
  ./Tasks

ДОСТУПНЫЕ НАПИТКИ (из data/recipes.txt)
-----------------------------------------
- Латте             (кофе × 1, молоко × 2)
- Капучино          (кофе × 2, молоко × 2)
- Американо         (кофе × 1, вода × 1)
- Эспрессо          (кофе × 1)
- Кофе раф          (кофе × 1, сливки × 1, ванильный сахар × 1)
- Флет Вайт         (кофе × 2, молоко × 1)
- Латте с бренди    (кофе × 1, молоко × 2, бренди × 1)
- Латте с корицей   (кофе × 1, молоко × 2, корица × 1)
- Капучино с сиропом (кофе × 2, молоко × 2, сироп × 1)
- Раф с корицей     (кофе × 1, сливки × 1, ванильный сахар × 1, корица × 1)

Чтобы добавить новый напиток, достаточно дописать его рецепт в
data/recipes.txt в указанном формате — перекомпиляция не требуется.
